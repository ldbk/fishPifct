fishPifct
=====

fishPifct is an R package combining some functions used in the fishPi project
([fishPi](<http://fishPi/link/>)).

#Installation

In a R console:

`install.packages("devtools")`

`library(devtools)`

`install_github("ldbk/fishPifct")`
